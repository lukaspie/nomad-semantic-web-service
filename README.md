# `nomad-semantic-web-service`: ESRF-style public dataset catalogue for NOMAD

`nomad-semantic-web-service` is a NOMAD plugin that finds and extracts public
synchrotron dataset records from ESRF-style catalogues, with semantic OpenAPI
annotations and PANET/ESRFET ontology term mapping.

This `nomad` plugin was generated with [`cookiecutter`](https://github.com/cookiecutter/cookiecutter) using the [`cookiecutter-nomad-plugin`](https://github.com/FAIRmat-NFDI/cookiecutter-nomad-plugin) template, and is built on top of [`oscarsSemanticWebService`](https://github.com/gkoum/oscarsSemanticWebService) by Giannis Koumoutsos.

## Docs

More information about this plugin is available in the [documentation](https://fairmat-nfdi.github.io/nomad-semantic-web-service/). You will find information about getting started, how-to guides, the REST API and ELN schema reference, and much more there.
